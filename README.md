# Neon & Memory: 2700年のノスタルジア

マルチエンディング型SFテキストアドベンチャーゲーム。

## 概要

西暦2700年の日本を舞台に、記憶生命体である主人公の選択を通じて人類の進化の結末を見届ける物語。
16bit風の緻密なピクセルアートとサイバーパンクな没入感のある表現をバニラHTML/CSS/JSで実装しています。
